  • Available must be: Yes or No (so you can mark leave/exemption right in the sheet)
  • Other column names are flexible — the app auto-detects them

Step 2: Publish the sheet

  1. In your Google Sheet, click File → Share → Publish to web
  2. In the dropdown, select "Comma-separated values (.csv)"
  3. Click Publish → confirm
  4. Copy the URL it gives you (looks like https://docs.google.com/spreadsheets/d/e/.../pub?output=csv)

Step 3: In the app

  1. Paste the URL into the input box
  2. Click 💾 Save (the URL is remembered in your browser forever)
  3. Click 🔄 Sync Now to fetch the staff list
  4. Every month: just update your sheet → click Sync
⚠️ Anyone with the published link can read your sheet. Don't put confidential info in it.

📖 Setup Guide — Settings Google Sheet

Step 1: Your sheet already has 3 tabs (Trackers, PhoneMapping, Template) ✓

Step 2: Paste the Apps Script bridge

  1. Open your Settings Google Sheet
  2. Click Extensions → Apps Script
  3. Delete any existing code
  4. Paste the settings_bridge.gs code (downloaded separately)
  5. Click 💾 Save → run setupSheets once

Step 3: Deploy as Web App

  1. In Apps Script → Click Deploy → New deployment
  2. Type: Web App
  3. Execute as: Me
  4. Who has access: Anyone
  5. Click Deploy → copy the Web App URL
  6. Paste into Apps Script Web App URL box in the webapp → Save

Step 4: Publish sheet for reading

  1. In your Settings Sheet → File → Share → Publish to web
  2. Entire Document → Comma-separated values (.csv) → Publish
  3. Copy the URL → paste into Published CSV URL box → Save

Step 5: On any other device (phone/iPad/office PC)

  1. Open the GitHub URL in your browser
  2. Enter PIN → go to 📲 WhatsApp Reminder tab
  3. Paste BOTH URLs → Save → Sync
  4. All trackers and phone mapping appear automatically
⚠️ The two URLs are saved in your browser's localStorage per device. Paste them once per device — then everything syncs automatically from Google Sheet.
